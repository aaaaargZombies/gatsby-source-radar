# gatsby-source-starter

This is a starter plugin for building a Gatsby JS Source Plugin. It helps to update this as you go.

## Set The Config

In `gatsby-config.js`:

```js
module.exports = {
  plugins: [
    {
      resolve: 'gatsby-source-starter',
      options: {},
    },
  ],
};
```
